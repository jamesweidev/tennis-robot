
# Tennis Robot Development

This repository documents my process of building the embedded, control, and vision subsystems for a tennis ball retrieving robot.

## Subsystems
Each subsystem will have its own project located in 'projects/', all of which will be combined eventually into an integrated prototype.
- Autonomous Mobile Base
- Vision-Based Tennis Ball Tracking
- Launcher Subsystem
- Ball Feeder
- Integrated Prototype

## Current Focus

Currently working on a basic autonomous base using an ultrasonic sensor, cameras will be used later. 

## Firmware

All firmware uses HAL are located in their respective project directory
